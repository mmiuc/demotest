# 前后端简单项目

这是一个简单的前后端项目，前端表格从后端获取数据。

## 项目结构
- `backend/`：后端代码目录
  - `server.js`：Node.js 后端服务
  - `package.json`：后端依赖文件
- `frontend/`：前端代码目录
  - `index.html`：前端页面，包含表格展示数据

## 运行步骤
### 后端
1. 进入后端目录：`cd backend`
2. 安装依赖：`npm install`
3. 运行后端服务：`npm start`

### 前端
直接在浏览器中打开 `frontend/index.html` 文件即可查看表格数据。
